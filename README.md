# RecA-TB QSAR Machine Learning Project

This repository contains a computational drug discovery workflow for
Mycobacterium tuberculosis RecA inhibitor prediction using:

- QSAR modeling
- Machine learning
- Molecular docking
- SHAP interpretation
- ADMET analysis
- FDA drug screening

## Workflow

1. ChEMBL data collection
2. Data curation
3. Fingerprint generation
4. Feature selection
5. Machine learning modeling
6. SHAP analysis
7. FDA drug prediction
8. Molecular docking validation
