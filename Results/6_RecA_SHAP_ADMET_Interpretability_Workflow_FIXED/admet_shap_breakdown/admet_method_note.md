# ADMET-likeness note

This analysis adds an early ADMET-likeness triage before local explanation of candidate RecA-TB predictions.

Included:
- Lipinski-style oral-likeness violations
- Veber-style permeability/flexibility screen
- Egan-like polarity/lipophilicity screen

Limitations:
- This is not a full ADMET prediction platform.
- It does not replace pkCSM, SwissADME, ADMETlab, or experimental ADME assays.

Interpretation:
- Supportive = passes most heuristic filters.
- Borderline = one part of the profile needs caution.
- High risk = multiple oral-likeness/permeability alerts.
